# MyTutorial - 教程序列

这是一个综合性的教程序列，包含多个编程项目和语言学习教程。

## 项目列表

### ZigTutorial
Zig 编程语言教程，包含从基础到高级的各种概念和示例代码：
- 基础语法（变量、常量、数据类型）
- 控制流（if语句、循环、switch）
- 函数与错误处理
- 指针和内存管理
- 结构体、枚举、联合体
- 高级特性（异步、原子操作、位操作）
- 构建系统和测试
- 实际项目示例

### SqlsugarTest
SqlSugar ORM 测试项目，包含：
- 配置管理
- 实体定义
- 服务层实现

## 学习目标

- 掌握多种编程语言的基础和高级特性
- 理解不同编程范式和设计模式
- 实践软件开发最佳实践
- 构建可维护、可扩展的代码

## 使用方法

每个项目都是独立的，可以单独学习和运行。请参考各项目目录下的具体说明文档。

## 许可证

本项目仅供学习使用。
