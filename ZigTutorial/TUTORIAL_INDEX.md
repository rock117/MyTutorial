# Zig 语言完整教程目录

## 📚 教程说明

本教程系列涵盖 Zig 语言从入门到精通的所有知识点，每个文件都是可独立运行的示例代码，带有详细的中文注释。

### 使用方法
```bash
# 运行某个教程文件
zig run 001_hello_world.zig

# 编译某个教程文件
zig build-exe 001_hello_world.zig

# 查看某个教程文件
cat 001_hello_world.zig
```
